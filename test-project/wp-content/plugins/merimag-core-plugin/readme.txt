== Description ==

Merimag Core Plugin Extend theme functionalities with amazing features

== Installation ==

1. In your admin panel, go to Plugings > click the Add New button.
2. Click Upload Plugin and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new plugin right away.
